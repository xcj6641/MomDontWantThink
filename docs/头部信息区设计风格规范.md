# 头部信息区设计风格规范

提取自：辅食常规页（todayFood）· 辅食计划页（week）

---

## 一、设计原则

- 头部信息区是页面的「情境锚点」，告诉用户「当前是谁的什么」
- 不堆砌信息，分层展示：主标题 → 副说明 → 辅助元数据
- 卡片本身有轻微浮起感（渐变背景 + 细边框 + 微阴影），与页面底色明确区分
- 右侧预留导航/操作链接入口，弱提示风格（下划线文字，不用按钮）

---

## 二、卡片容器规范

| 属性 | 值 |
|------|-----|
| 布局 | `flex`（icon 左 + 内容右）或 `grid 2列`（icon 占 max-content，内容占 1fr） |
| 列间距 | `20rpx` |
| 背景 | 浅绿渐变：`linear-gradient(165–168deg, #fbfdf9 0%, #eef6f1 40–100%, #dff5ea 100%)`（起止角度因页面略异，保持同系） |
| 边框 | `2rpx solid rgba(160, 216, 192, 0.38–0.40)` |
| 圆角 | `32rpx` |
| 内边距 | `32rpx` |
| 外边距（下） | `32rpx` |
| 阴影 | `0 6rpx 20rpx rgba(47, 94, 59, 0.05)`（可选，辅食常规页有，计划页无） |
| box-sizing | `border-box` |

---

## 三、元素层级与字体规范

### 3.1 左侧 Icon（Emoji）

| 属性 | 值 |
|------|-----|
| font-size | `44rpx` |
| line-height | `1` |
| flex-shrink | `0`（不压缩） |
| 垂直对齐 | `align-items: flex-start`（顶部对齐） |

> 示例：辅食常规页用 👶，计划页用 🌱

---

### 3.2 主标题（第一层）

| 属性 | 值 |
|------|-----|
| font-size | `34rpx` |
| font-weight | `600` |
| color | `var(--text-heading)` → `#2f5e3b` |
| margin-bottom | `8rpx` |
| display | `block` |

> 示例：「宝宝 · 10个月」/ 「本周辅食准备好啦」

---

### 3.3 副标题 / 鼓励语（第二层）

| 属性 | 值 |
|------|-----|
| font-size | `26rpx` |
| font-weight | `400` |
| color | `#6d8474` 或 `var(--text-secondary)` → `#6e7b70`（两者接近，统一使用 `#6e7b70`） |
| line-height | `1.45` |
| display | `block` |

> 示例：「今天也要好好吃饭呀 🌱」/ 「根据宝宝10月龄生成，可以随时调整」

---

### 3.4 元数据行（第三层，分隔线上方）

适用于需要展示参考摄入量、过敏食材等补充信息的场景。

| 属性 | 值 |
|------|-----|
| font-size | `24rpx` |
| font-weight | `400` |
| color | `var(--text-muted)` → `#7d8b80` |
| line-height | `1.45` |
| 与上方分隔 | `margin-top: 16rpx`；`padding-top: 16rpx`；`border-top: 1rpx solid rgba(160, 216, 192, 0.35)` |

---

### 3.5 右侧导航链接

| 属性 | 值 |
|------|-----|
| font-size | `24rpx` |
| font-weight | `500` |
| color | `var(--green-deep)` → `#6ba86a` |
| 下划线 | `border-bottom: 1rpx solid rgba(107, 168, 106, 0.45)` |
| padding-bottom | `2rpx` |
| line-height | `1.35` |
| 点击态 | `opacity: 0.55` |
| 布局 | 纵向排列（`flex-direction: column`），靠右（`align-items: flex-end`），`gap: 12rpx` |

> 示例：「本周计划 >」「查看下周 >」

---

## 四、页面内嵌标题行（卡片内 section header）

适用于卡片内部区块标题（如「今日辅食」标题行），与上方信息卡有所区分。

| 属性 | 值 |
|------|-----|
| 布局 | `flex`，`justify-content: space-between`，`align-items: center` |
| Emoji 图标 | `font-size: 32rpx`，`line-height: 48rpx` |
| 标题文字 | `font-size: 34rpx`，`font-weight: 600`，`color: var(--text-heading)`，`line-height: 48rpx` |
| 副标题 | `font-size: 26rpx`，`color: var(--text-secondary)`，`line-height: 1.45` |
| 右侧操作 | `font-size: 26rpx`，`font-weight: 500`，`color: var(--green-deep)`；下划线 `border-bottom: 1rpx solid rgba(127, 183, 126, 0.45)` |
| 与下方内容分隔 | `border-bottom: 2rpx solid var(--border-warm)`，`padding-bottom: 24rpx` |

---

## 五、CSS 变量速查

| 变量 | 值 | 用途 |
|------|-----|------|
| `--bg-warm` | `#f7faf8` | 页面背景 |
| `--card` | `#ffffff` | 普通卡片背景 |
| `--green` | `#7fb77e` | 主绿色（按钮、进度条） |
| `--green-deep` | `#6ba86a` | 深绿（链接、强调） |
| `--text-heading` | `#2f5e3b` | 主标题文字 |
| `--text-body` | `#2f3a35` | 正文文字 |
| `--text-secondary` | `#6e7b70` | 副标题、说明文字 |
| `--text-muted` | `#7d8b80` | 元数据、弱提示文字 |
| `--border-warm` | `#e4ede6` | 卡片边框、分隔线 |
| `--shadow-card` | `0 12rpx 40rpx rgba(127,183,126,0.07), 0 4rpx 16rpx rgba(90,120,95,0.05)` | 标准卡片阴影 |

## 七、适用范围

- ✅ 辅食常规页（todayFood）宝宝信息卡 `baby-bar`
- ✅ 辅食计划页（week）状态提示卡 `welcome-card`
- 🔜 本周备餐计划页（weekSummary）头部区参照此规范统一
- 🔜 其他新页面头部信息区默认遵循此规范
