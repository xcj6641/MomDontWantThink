# 本周备餐计划页（weekSummary）

页面文件：`miniprogram/pages/weekSummary/`
入口：todayFood 页宝宝信息栏「本周计划 >」/「查看下周计划」按钮（已实现）

---

## 页面定位

执行型页面，不是展示型页面。用户通常在收到"周末备菜提醒"后进入，照着清单完成本周备菜。
页面核心：**让妈妈照着清单完成本周备菜**，不重复展示宝宝信息。

---

## 页面结构（5区）

### 1. 头部信息区 ✅
- 导航栏标题：`本周备餐计划` / `下周备餐计划`（由 `isNextWeek` 参数控制）
- 遵循「头部信息区设计风格规范」（`docs/头部信息区设计风格规范.md`）：
  - 卡片：渐变浅绿背景 `linear-gradient(165deg, #fbfdf9, #eef6f1)`，细绿边框，`32rpx` 圆角，`32rpx` 内边距
  - 左侧 Icon：`<view>` 元素（非 `<text>`）内放 `📅`，`44rpx line-height:1`，与标题顶部对齐；`info-card-top` 使用 `align-items: flex-start` 确保图标与主标题始终顶部对齐
  - 主标题：`备餐周期` `34rpx 600 #2f5e3b`
  - 副标题：日期范围 `26rpx #6e7b70`
  - 右侧操作区（`info-card-actions`，纵向排列，右对齐）：
    - 收藏按钮（`fav-btn`）：行内布局（`gap: 6rpx`，`min-width: 184rpx` 锁定宽度防止切换时 icon 位移），图片 icon（`32×32rpx`，`/assets/star.png` / `/assets/star_active.png`，`mode="aspectFit"`）+ 文字标签（`24rpx 500 #6ba86a`，无下划线）；未收藏：`star.png` + `收藏周计划`；已收藏：`star_active.png` + `已收藏周计划`；点击切换 `isFavorited` 状态并弹出 Toast
    - 导航链接：`查看/调整本周计划` `24rpx #6ba86a`，下划线样式，点击态 `opacity: 0.55`

### 2. 完成进度区 ✅
- 文案：`已完成 X / Y 项` + 百分比（30rpx 700，深绿 #2d6e37 强调）
- 进度条高度 20rpx，随勾选实时更新；进度条前端（填充边缘）有 🥑 牛油果 icon 随进度移动
- **统计范围**：食材清单项 + 提前准备项（不含备餐详情折叠区）
- 卡片背景：浅绿色 `#EAF6EC`，无阴影，突出"重点完成"感

### 3. 食材清单区 ✅
- 区块标题左侧有 🛒 icon
- 从本周菜单自动汇总，相同食材合并展示总量
- 按类别分组，每组一张卡片：蔬菜类 / 蛋白质类 / 水果类 / 主食类 / 其他（**类别行不显示 icon**）
- 每项：左侧 checkbox + 食材名 + 数量
- 勾选状态：未完成正常色；已完成：checkbox 主题绿、文字浅绿色（`#8fbe8e`）、无删除线、无整体半透明
- 进度同步更新，可取消
- 区块标题右侧有「全部标记为已完成」文本按钮，**仅标记食材清单项**，进度同步更新

### 4. 提前准备区 ✅
- 模块名称：🥄 **提前准备**（副说明：建议周末完成）；标题右侧有「全部已完成」文本按钮，**仅标记提前准备项**，进度同步更新
- 动作清单，每项：checkbox + 准备动作名称
- 每项下方有「用于：N道菜」折叠入口，点击 inline 展开关联菜品列表（不使用弹窗）
- 勾选参与整体进度统计

### 5. 📋 本周备餐计划详情区 ✅
- 折叠面板（Accordion），默认全部折叠，手风琴模式（同时只展开一个）
- 折叠态：轮次名（备餐一/二/三）+ 对应日期 + 摘要（X道菜）+ 箭头
- 展开后（仅展示备餐模板，不按日期拆分）：
  - 按餐次分组：早餐 / 午餐 / 晚餐
  - 每个餐次下列出对应菜品
  - 若该轮次内某天存在单日调整，用 `/` 表示差异：`默认菜 / 替换菜`
  - 不展示每日拆分、提前准备内容、对应餐次说明
  - 不提供单个菜品点击跳转
